# Alien Invasion 👾

A simple 2D arcade-style shooter game built with **Python** and **Pygame**.  
The player controls a spaceship and must shoot down waves of invading aliens while avoiding losing all ships.

---

## Features

- 🚀 Player-controlled spaceship
- 👾 Alien fleet generation
- 🔫 Bullet shooting system
- 💥 Collision detection
- 🧮 Score tracking
- ❤️ Remaining ships display
- 🎮 Keyboard controls

---

## Controls

| Key | Action |
|----|------|
| Left Arrow | Move ship left |
| Right Arrow | Move ship right |
| Spacebar | Shoot bullets |
| Q | Quit the game |

---

## Installation

1. Clone the repository


- ` git clone https://github.com/AhmedKhairy22/alien-invasion.git`

 - `cd alien-invasion`


2. Install Pygame


- `pip install pygame`


---

## Run the Game


`python alien_invasion.py`
